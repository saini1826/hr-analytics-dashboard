## HR Analytics Dashboard

## Overview
This HR Analytics Dashboard analyzes employee attrition, salary distribution, job roles, and workforce trends to support data-driven HR decisions.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel / CSV

## KPIs
- Total Employees: 1470
- Attrition: 237
- Attrition Rate: 16.1%
- Avg Age: 37
- Avg Salary: 6.5K

## Key Insights
- Highest attrition in Sales and Laboratory Technician roles
- Employees with salary below 5K have higher attrition
- Age group 26–35 shows maximum attrition

## Features
- Department-wise analysis
- Salary analysis
- Attrition by age, gender, and education
- Interactive filters and charts

## Conclusion
This dashboard helps HR teams identify attrition trends and improve workforce decision-making.
